Linux Log Monitor with Email Alerts

A lightweight Python-based log monitoring tool for Linux systems that detects and alerts you about suspicious activity in real-time. It helps track system misuse and alerts you proactively.


Key Features:

• Detect failed login attempts from /var/log/auth.log
• Monitor usage of sudo commands
• Track the creation of new user accounts
• Instantly send email alerts to your configured address
• Log all alerts to a file: logs/suspicious_activity.log
• Fully automatable using cron (every 15 minutes)


Technology used: Python, Linux, Cron, SMTP
